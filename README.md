# SnookerDataAnalysis
Personal snooker training data analysis project built with Python.

## Project Introduction
This project records my daily amateur snooker training data and uses data statistics & visualization to analyze training performance. All data is collected from my personal 8-foot practice table.

## Technology Stack
- Python
- Pandas: Data sorting and statistical calculation
- Matplotlib: Generate shot rate charts
- Numpy: Numerical data processing

## Existing Script Files
- `0621_input_snooker.py`: Data input script for training on 2026-06-21
- `training_record_0622.py`: Complete shot statistics script for 2026-06-22 practice

## Core Tracking Metrics
1. Long pot hit rate
2. 5-point positional shot success rate
3. Training fatigue & error distribution statistics

## Future Improvement Plan
1. Automatically generate weekly training performance visual reports
2. Build long-term trend comparison of shooting accuracy
3. Simplify manual data entry process
